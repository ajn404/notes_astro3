#### [Array.prototype.toSorted()](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/toSorted)

```js
// 不传入函数
toSorted();
// 传入箭头函数
toSorted((a, b) => {
  /* … */
});
// 传入比较函数
toSorted(compareFn);
// 內联比较函数
toSorted(function compareFn(a, b) {
  /* … */
});
```

        <div class="text-skin-blue">Array 实例的 toSorted() 方法是 sort() 方法的复制方法版本。它返回一个新数组，其元素按升序排列。</div>

#### [npm link](https://docs.npmjs.com/cli/v8/commands/npm-link)

> npm link 是一个 npm 命令，用于在本地开发中创建软链接，将一个本地模块链接到另一个正在开发的项目中。<br/>
> 当你在开发一个 npm 模块时，可以使用 npm link 将该模块链接到其他项目中进行测试和调试。这样做可以避免每次修改模块代码后都需要手动重新安装模块。<br />
> 使用 npm link 的步骤如下：

<div class="text-skin-purple">
  1.在要链接的模块的根目录中运行 npm link 命令。这将在全局 node_modules
  目录中创建一个指向该模块的软链接。
  <br />
  2.在要使用该模块的项目中，运行 npm link 模块名 命令。这将在项目的 node_modules
  目录中创建一个指向全局 node_modules 目录中的软链接。
</div>

#### [promise practice](https://www.codingame.com/playgrounds/347/javascript-promises-mastering-the-asynchronous/more-pratice-with-promises)

```js
function job(data) {
  return new Promise((resolve, reject) => {
    if (typeof data !== "number") {
      reject("error");
    } else if (data % 2 === 0) {
      setTimeout(() => {
        reject("even");
      }, 2000);
    } else {
      setTimeout(() => {
        resolve("odd");
      }, 1000);
    }
  });
}

module.exports = job;
```
